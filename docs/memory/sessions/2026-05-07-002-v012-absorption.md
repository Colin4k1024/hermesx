## 关键决策

1. **Copy-on-read for LifecycleHooks** — RLock + copy slice before executing hooks, avoids holding lock during slow hook callbacks
2. **Rune-aware truncation** — All LLM prompt sanitization uses `[]rune` to avoid invalid UTF-8 sequences
3. **Explicit sort before eviction** — Never rely on store.List() ordering for business logic; sort at call site
4. **Sentinel errors for input validation** — ErrMissingChatID + ErrInvalidPath enable typed error assertions in tests
