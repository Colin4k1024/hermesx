# 开源治理

本页汇总 HermesX 的公开协作入口。根目录文件是 GitHub 上的正式入口，文档站保留导航入口，避免新贡献者在 README、issue 模板和文档站之间迷路。

| 主题 | 入口 |
|------|------|
| 贡献流程 | [CONTRIBUTING.md](https://github.com/Colin4k1024/hermesx/blob/main/CONTRIBUTING.md) |
| 行为准则 | [CODE_OF_CONDUCT.md](https://github.com/Colin4k1024/hermesx/blob/main/CODE_OF_CONDUCT.md) |
| 安全政策 | [SECURITY.md](https://github.com/Colin4k1024/hermesx/blob/main/SECURITY.md) |
| 支持入口 | [SUPPORT.md](https://github.com/Colin4k1024/hermesx/blob/main/SUPPORT.md) |
| 治理说明 | [GOVERNANCE.md](https://github.com/Colin4k1024/hermesx/blob/main/GOVERNANCE.md) |
| 路线图 | [ROADMAP.md](https://github.com/Colin4k1024/hermesx/blob/main/ROADMAP.md) |
| 许可证 | [LICENSE](https://github.com/Colin4k1024/hermesx/blob/main/LICENSE) |

安全漏洞不要在公开 issue 中披露细节。请按 [SECURITY.md](https://github.com/Colin4k1024/hermesx/blob/main/SECURITY.md) 使用私密漏洞上报或先开非敏感联系 issue。
