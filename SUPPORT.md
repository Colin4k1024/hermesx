# Support / 支持

## English

Use the channel that matches the kind of help you need:

| Need | Channel |
|------|---------|
| Bug report with reproduction steps | [Bug report](https://github.com/Colin4k1024/hermesx/issues/new/choose) |
| Setup or build help | [Setup / Build Help](https://github.com/Colin4k1024/hermesx/issues/new/choose) |
| Feature request or design proposal | [Feature request](https://github.com/Colin4k1024/hermesx/issues/new/choose) |
| Security vulnerability | Follow [SECURITY.md](SECURITY.md); do not disclose details publicly |

Before opening an issue, please check the [README](README.md), [documentation site](docs/index.en.md), and existing issues. For setup problems, include the exact command, operating system, Go version, and full error output.

HermesX is open source software. Support is best effort unless you have a separate support agreement with a maintainer or vendor.

## 中文

请按问题类型选择支持入口：

| 需求 | 入口 |
|------|------|
| 带复现步骤的缺陷报告 | [Bug report](https://github.com/Colin4k1024/hermesx/issues/new/choose) |
| 安装、构建或配置帮助 | [Setup / Build Help](https://github.com/Colin4k1024/hermesx/issues/new/choose) |
| 功能建议或设计提案 | [Feature request](https://github.com/Colin4k1024/hermesx/issues/new/choose) |
| 安全漏洞 | 遵循 [SECURITY.md](SECURITY.md)，不要公开披露漏洞细节 |

提交 issue 前，请先查看 [README](README.md)、[文档入口](docs/index.md) 和已有 issue。安装或构建问题请提供完整命令、操作系统、Go 版本和完整错误输出。

HermesX 是开源软件。除非你与维护者或供应方另有支持协议，否则支持以社区 best effort 为准。
